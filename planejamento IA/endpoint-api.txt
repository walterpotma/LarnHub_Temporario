1. Autenticação e Usuários

Autenticação

POST /api/auth/register - Registrar novo usuário
POST /api/auth/login - Fazer login
POST /api/auth/refresh - Refresh token
POST /api/auth/logout - Fazer logout
POST /api/auth/forgot-password - Solicitar reset de senha
POST /api/auth/reset-password - Resetar senha com token

Usuários

GET /api/users/me - Obter dados do usuário logado
PUT /api/users/me - Atualizar perfil do usuário
DELETE /api/users/me - Desativar conta
GET /api/users/{id} - Obter dados de um usuário (admin)
PUT /api/users/{id} - Atualizar usuário (admin)
GET /api/users - Listar usuários (admin, com paginação)

2. Cursos

Cursos

GET /api/courses - Listar cursos (com filtros e paginação)
POST /api/courses - Criar curso (instrutor/admin)
GET /api/courses/{id} - Obter detalhes de um curso
PUT /api/courses/{id} - Atualizar curso (instrutor/admin)
DELETE /api/courses/{id} - Remover curso (admin)
GET /api/courses/{id}/students - Listar estudantes do curso (instrutor/admin)
GET /api/courses/popular - Listar cursos populares
GET /api/courses/recommended - Listar cursos recomendados (baseado no histórico)

Módulos e Aulas

GET /api/courses/{courseId}/modules - Listar módulos de um curso
POST /api/courses/{courseId}/modules - Criar módulo (instrutor/admin)
GET /api/courses/{courseId}/modules/{moduleId} - Obter módulo
PUT /api/courses/{courseId}/modules/{moduleId} - Atualizar módulo (instrutor/admin)
DELETE /api/courses/{courseId}/modules/{moduleId} - Remover módulo (instrutor/admin)

GET /api/courses/{courseId}/modules/{moduleId}/lessons - Listar aulas de um módulo
POST /api/courses/{courseId}/modules/{moduleId}/lessons - Criar aula (instrutor/admin)
GET /api/courses/{courseId}/modules/{moduleId}/lessons/{lessonId} - Obter aula
PUT /api/courses/{courseId}/modules/{moduleId}/lessons/{lessonId} - Atualizar aula (instrutor/admin)
DELETE /api/courses/{courseId}/modules/{moduleId}/lessons/{lessonId} - Remover aula (instrutor/admin)

3. Inscrições e Progresso

Inscrições

POST /api/enrollments - Inscrever-se em um curso
GET /api/enrollments - Listar inscrições do usuário
GET /api/enrollments/{id} - Obter detalhes de uma inscrição
DELETE /api/enrollments/{id} - Cancelar inscrição (com políticas)
GET /api/enrollments/{id}/progress - Obter progresso no curso

Progresso

POST /api/progress - Marcar aula como concluída
GET /api/progress - Listar progresso do usuário
GET /api/progress/{courseId} - Obter progresso em um curso específico

4. Avaliações e Comentários

Avaliações

POST /api/courses/{courseId}/reviews - Criar avaliação
GET /api/courses/{courseId}/reviews - Listar avaliações do curso
GET /api/reviews/{id} - Obter avaliação específica
PUT /api/reviews/{id} - Atualizar avaliação
DELETE /api/reviews/{id} - Remover avaliação

Comentários

POST /api/lessons/{lessonId}/comments - Adicionar comentário
GET /api/lessons/{lessonId}/comments - Listar comentários da aula
PUT /api/comments/{id} - Atualizar comentário
DELETE /api/comments/{id} - Remover comentário

5. Certificados

GET /api/certificates - Listar certificados do usuário
GET /api/certificates/{id} - Obter certificado específico
POST /api/courses/{courseId}/certificates - Gerar certificado (quando curso concluído)
GET /api/certificates/{id}/download - Download do certificado em PDF
GET /api/certificates/{id}/validate - Validar certificado (público)

6. Pagamentos e Transações

POST /api/payments/intent - Criar intenção de pagamento
GET /api/payments/methods - Listar métodos de pagamento do usuário
POST /api/payments/methods - Adicionar método de pagamento
DELETE /api/payments/methods/{id} - Remover método de pagamento
GET /api/transactions - Listar transações do usuário
GET /api/transactions/{id} - Obter detalhes de transação

7. Administração

Categorias

GET /api/categories - Listar categorias
POST /api/categories - Criar categoria (admin)
PUT /api/categories/{id} - Atualizar categoria (admin)
DELETE /api/categories/{id} - Remover categoria (admin)

Instrutores

GET /api/instructors - Listar instrutores
POST /api/instructors - Aprovar instrutor (admin)
GET /api/instructors/{id} - Obter perfil de instrutor
PUT /api/instructors/{id} - Atualizar perfil de instrutor
GET /api/instructors/{id}/courses - Listar cursos do instrutor

Relatórios

GET /api/admin/reports/enrollments - Relatório de inscrições
GET /api/admin/reports/revenue - Relatório de receita
GET /api/admin/reports/course-performance - Desempenho dos cursos

8. Recursos Adicionais

Uploads

POST /api/uploads - Upload de arquivos (imagens, vídeos, documentos)
GET /api/uploads/{id} - Obter arquivo
DELETE /api/uploads/{id} - Remover arquivo

Notificações

GET /api/notifications - Listar notificações do usuário
PUT /api/notifications/{id}/read - Marcar como lida
GET /api/notifications/unread - Contar notificações não lidas







Especificações Técnicas
Padrões Adotados:
Autenticação: JWT (Bearer token)

Paginação: ?page=1&limit=10

Filtros: ?category=programming&level=beginner

Ordenação: ?sort=-createdAt (desc) ou ?sort=price (asc)

Busca: ?q=javascript

Formatos: JSON para requests e responses

Status Codes apropriados para cada operação

Versionamento: /api/v1/ para futuras atualizações
